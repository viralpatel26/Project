Authors: Viral Patel, Justin Van Groningen, Khushi Sunil Bhatt

About the program: The code enclosed in this program is used to play an online battleship game. The two players communicate with each other via the live chat option.

How to run the program: Unzip the given file and locate the packages in it.Open up a project in IDE. Open up the ‘Server’ package and run ‘server.java’. Now navigate back to the ‘Client’ package and run ‘client.java’ twice.
